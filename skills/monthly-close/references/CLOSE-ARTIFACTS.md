# Monthly Close Artifacts

Use this file before declaring a month closed.

## Required outputs

- reconciliation report for each bank and credit account
- monthly P&L
- list of adjusting entries
- outstanding items carried forward
- exception queue for unresolved items

## Do not mark the month closed if

- any core cash account is unreconciled
- uncategorized transactions remain material
- open exceptions could change revenue, expenses, owner equity, payroll, or tax treatment
- source documentation is still missing for material transactions

## Approval boundary

Require explicit user approval before:

- locking a period in accounting software
- posting adjusting entries that affect taxes, payroll, or owner equity
- closing a month with unresolved material exceptions
