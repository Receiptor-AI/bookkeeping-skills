# Tax Package Requirements

Use this file before delivering a tax package to an accountant or filing workflow.

## Package should include

- full-year P&L
- quarterly breakdown
- expense detail with source references
- reconciliation status for year-end accounts
- contractor summary and 1099 status
- schedules for depreciation, vehicle, meals, and home office when applicable

## Escalate before delivery when

- books are not reconciled through year-end
- material expenses lack source documentation
- special-treatment areas are incomplete
- estimated tax, payroll, owner draw, or entity questions remain unresolved

## Approval boundary

Do not present the package as final for filing unless the user approves the package and understands that the skill organizes data rather than giving legal or tax advice.
